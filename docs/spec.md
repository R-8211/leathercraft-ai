# LeatherCraft AI

## 概要

LeatherCraft AI は、レザークラフト初心者〜中級者を対象とした、AI対話型の革小物型紙自動生成 Web アプリケーションである。ユーザーは「マチ付きの二つ折り財布、カードは6枚入る感じで」のように日本語で要望を伝えるだけで、寸法・縫い代・切り込み線を含む実寸 SVG 型紙が即座に生成され、A4 用紙で家庭用プリンター印刷可能な PDF としてダウンロードできる。MVP では財布系（二つ折り・三つ折り・長財布・カードケース）と小物系（小銭入れ・コインケース・キーケース）に対応する。フリーミアムモデルで、無料枠は月 5 回・透かし入り、有料プランは月額 480 円〜で月 50 回または無制限となる。

## スプリント計画

### Sprint 1: プロジェクト基盤とランディング

**ゴール:** 誰でもアクセスできるトップ画面を立ち上げ、プロダクトの価値提案を伝える。

**機能:**
- [ ] ランディングページ: ヒーロー領域に「日本語で伝えるだけで型紙が手に入る」コピー、サンプル型紙画像、CTA ボタンを配置
- [ ] ナビゲーション: トップ / 使い方 / 料金 / ログインのグローバルナビ
- [ ] 料金プランページ: Free / Hobby / Creator / Pro の比較表
- [ ] フッター: 利用規約・プライバシーポリシー・運営情報のリンクプレースホルダ
- [ ] レスポンシブ対応: スマホ・タブレット・デスクトップで崩れないレイアウト

**受け入れ基準:**
- ユーザーがランディングページにアクセスすると 3 秒以内にヒーローが表示されること
- CTA ボタンを押すとアイテム選択画面へ遷移すること（未ログインでも体験可）
- スマホ幅（375px）で横スクロールが発生しないこと

---

### Sprint 2: アイテム選択とチャット UI 骨格

**ゴール:** ユーザーが作りたいアイテムを選び、日本語で要望を入力する一連の体験の土台を作る。

**機能:**
- [ ] アイテム選択画面: 「二つ折り財布」「三つ折り財布」「長財布」「カードケース」「小銭入れ」「コインケース」「キーケース」のカード一覧。各カードにイラスト・難易度・必要時間目安を表示
- [ ] チャット画面レイアウト: 左側にチャット履歴、右側に型紙プレビュー枠（初期は空状態のプレースホルダ）の 2 ペイン構成
- [ ] メッセージ入力: テキスト入力欄、送信ボタン、Enter キー送信、Shift+Enter で改行
- [ ] AI 応答プレースホルダ: 入力に対して「考え中...」のローディング表示と、ダミー応答が返るスタブ実装
- [ ] サジェスト例: 「カードが6枚入る薄めの財布」「小銭がたくさん入る box 型コインケース」など、初心者向けの入力例を 3 つ表示

**受け入れ基準:**
- アイテムをクリックするとチャット画面に遷移し、選択アイテム名が画面上部に表示されること
- メッセージを送信すると、ユーザー発言が右寄せ吹き出しで履歴に追加されること
- サジェスト例をクリックすると、その文字列が入力欄に挿入されること

---

### Sprint 3: AI 対話と要件抽出

**ゴール:** ユーザーの自然言語入力から、型紙生成に必要な構造化要件を AI が抽出する。

**機能:**
- [ ] AI チャット応答: ユーザー入力を Claude へ送り、要件不足があれば追質問する対話ロジック
- [ ] 要件抽出: 「幅」「高さ」「マチ」「カードポケット数」「小銭入れの有無」「角丸 R」など、アイテム別のパラメータを構造化して保持
- [ ] 要件確認カード: 抽出した値をチャット内にカード形式で表示し、「この内容で生成」ボタンを提示
- [ ] 値の手動編集: 確認カード内の各値をその場で編集可能（数値入力 / トグル）
- [ ] 不明値のデフォルト補完: ユーザーが指定しなかった項目には標準的な推奨値を補完し、補完であることを明示
- [ ] 入力バリデーション: 極端な寸法（例: 幅 5mm、高さ 1m）に対する警告メッセージ

**受け入れ基準:**
- ユーザーが「カードが 6 枚入る二つ折り財布」と入力すると、幅・高さ・カード枚数 6 が確認カードに表示されること
- 寸法が不明な場合に AI が「縦のサイズはどのくらいにしますか？」のように追質問すること
- 確認カードの値を編集して「生成」ボタンを押せること

---

### Sprint 4: SVG 型紙生成エンジン（小物系から）

**ゴール:** 構造化された要件から実寸 SVG 型紙を生成する。まずはパーツ数の少ない小銭入れ・コインケース・キーケースを対象とする。

**機能:**
- [ ] 小銭入れ（L 字ファスナー型・スナップ型）の型紙生成: 本体パーツ、ベロ、補強材の各 SVG パス
- [ ] コインケース（box 型）の型紙生成: 5 面展開図と折り目線
- [ ] キーケースの型紙生成: 本体、内装、金具取付穴位置
- [ ] 縫い代描画: 各パーツの周囲に指定幅（デフォルト 4mm）の縫い代ラインを破線で追加
- [ ] ステッチ穴ガイド: 縫い代ラインに沿って等間隔のステッチ穴位置を点で表示
- [ ] 折り目・コバ線: 折る位置を一点鎖線、コバ磨き対象辺をマークで明示
- [ ] パーツラベル: 「本体表」「内装」「ベロ」など各パーツに名称と「× 1 枚」などの数量を表記
- [ ] 寸法注記: 主要な辺の長さを mm 単位で SVG 上に表示

**受け入れ基準:**
- 小銭入れの要件を入力すると、5 秒以内に SVG が右ペインに表示されること
- 生成された SVG をブラウザで実寸表示したとき、定規で測った長さが指定値と一致すること（誤差 ±0.5mm 以内）
- 縫い代・ステッチ穴・折り目線が視覚的に区別できること（線種・色を変える）

---

### Sprint 5: SVG 型紙生成エンジン（財布系）

**ゴール:** パーツ数が多くマチ・カードポケットを含む財布類の型紙生成に対応する。

**機能:**
- [ ] 二つ折り財布の型紙生成: 外装、内装、カードポケット（指定枚数）、札入れマチ、小銭入れマチ
- [ ] 三つ折り財布の型紙生成: 三つ折り構造に対応した展開パーツ群
- [ ] 長財布の型紙生成: ラウンドファスナー型 / かぶせ型の選択、内装パーツ群
- [ ] カードケースの型紙生成: カード枚数に応じた段差付きカードポケット
- [ ] パーツ自動配置: 複数パーツを SVG キャンバス上で重ならないよう自動レイアウト
- [ ] パーツ間の整合性チェック: 縫い合わせる辺の長さが対応するパーツ間で一致すること
- [ ] 革厚補正: ユーザーが指定した革厚（1.0/1.2/1.5/2.0mm）に応じて折り返し部分の寸法を補正

**受け入れ基準:**
- 「カード 6 枚入りの二つ折り財布、革厚 1.5mm」を指定すると、外装・内装・カードポケット・マチが個別パーツとして生成されること
- カードポケット段差が指定枚数分積み重なる配置で表示されること
- 縫合する辺の長さが対応するパーツ間で完全一致すること（自動チェックで検証）

### Sprint 6: プレビュー操作と寸法調整

**ゴール:** 生成された型紙をユーザーが確認・調整できる UI を提供する。

**機能:**
- [ ] SVG プレビュー操作: マウスホイールでズーム、ドラッグでパン、ダブルクリックで全体表示
- [ ] パーツ単体表示切り替え: パーツ一覧から個別パーツのみを拡大表示
- [ ] 寸法調整パネル: 主要寸法（幅・高さ・マチ・カード枚数・角丸 R）のスライダーと数値入力
- [ ] 即時再生成: スライダー操作後、500ms のデバウンスで型紙を自動再生成
- [ ] 変更前後の比較: 「元に戻す」ボタンで直前の寸法に復帰
- [ ] パラメータプリセット保存: 現在の寸法セットに名前を付けて一時保存
- [ ] 縫い代幅・ステッチピッチ設定: グローバルオプションとして変更可能

**受け入れ基準:**
- スライダーで幅を変更すると 1 秒以内にプレビューが更新されること
- パーツ一覧から「カードポケット」をクリックすると、そのパーツのみが拡大表示されること
- 「元に戻す」を押すと直前の寸法状態に戻ること

### Sprint 7: PDF 出力

**ゴール:** A4 用紙で家庭用プリンター印刷できる、実寸スケールの PDF をダウンロードできるようにする。

**機能:**
- [ ] A4 自動分割: 型紙が A4 に収まらない場合、複数枚に自動分割
- [ ] つなぎ目マーク: 各 A4 ページの端に、隣接ページとの位置合わせ用のトンボ・矢印・ページ番号を印字
- [ ] 実寸スケール検証: 各ページに 50mm の検証用スケールバーを印字
- [ ] パーツ別ページ分け / 一括ページ: ユーザーが選択可能
- [ ] PDF メタ情報: ファイル名にアイテム名・主要寸法・生成日時を含む（例: `bifold-wallet-95x110_20260509.pdf`）
- [ ] 表紙ページ: 1 ページ目にアイテム概要、必要な革のサイズ、推奨革厚、パーツ一覧と数量を記載
- [ ] フリープラン透かし: 無料ユーザーには PDF に薄く「LeatherCraft AI Free」の斜め透かしを入れる
- [ ] ダウンロード履歴: 直近のダウンロード PDF を再ダウンロードできる

**受け入れ基準:**
- 「PDF ダウンロード」ボタンを押すと、3 秒以内にダウンロードが開始されること
- 印刷した A4 用紙の検証スケールバーを定規で測ると 50mm（誤差 ±0.5mm）であること
- 複数枚に分割された PDF を貼り合わせると、型紙の連続性が保たれること
- フリーユーザーの PDF には透かしが入り、有料ユーザーには入らないこと

### Sprint 8: アカウントと利用回数管理

**ゴール:** ユーザーアカウントを導入し、フリーミアム制の利用回数制限を機能させる。

**機能:**
- [ ] サインアップ / ログイン: メールアドレス + パスワード、または Google ログイン
- [ ] 未ログイン体験: ログインなしでも 1 回だけ生成体験できる（端末識別ベース）
- [ ] アカウントメニュー: ヘッダーからプロフィール・プラン・ログアウトにアクセス
- [ ] 利用回数カウンタ表示: ヘッダーに「今月 3 / 5 回」のように常時表示
- [ ] 上限到達時の挙動: フリープランで月 5 回到達後は生成ボタンが無効化され、アップグレード CTA を表示
- [ ] 月次リセット: 毎月 1 日に自動的にカウンタがリセットされる
- [ ] 履歴の同期: ログイン時、ローカル保存の型紙履歴をアカウントに紐付ける移行処理

**受け入れ基準:**
- 新規ユーザーがメール登録するとログイン状態でアイテム選択画面に遷移すること
- フリープランで 5 回生成すると 6 回目の生成ボタンが押せず、アップグレード画面への導線が表示されること
- 翌月 1 日にカウンタが 0 / 5 に戻ること

- [ ] 検索・フィルタ: アイテム種別・タグ・作成日でフィルタリング

## UI/UX 要件

### 画面構成

1. **ランディング** — ヒーロー、サンプル型紙ギャラリー、料金プラン、FAQ、CTA
2. **アイテム選択** — カードグリッド（イラスト・難易度・所要時間）
3. **チャット & プレビュー** — 左ペイン: チャット履歴と入力 / 右ペイン: SVG プレビューと寸法調整
4. **要件確認カード** — チャット内でインライン表示、編集・生成ボタン
5. **PDF プレビュー / ダウンロード** — モーダルで A4 分割プレビューを表示
6. **ライブラリ** — 型紙サムネイルのグリッド、タグ・検索
7. **アカウント / プラン** — 利用状況、アップグレード、決済履歴
8. **設定** — 縫い代幅・ステッチピッチ・革厚デフォルトなど

### ユーザーフロー（最短経路）

ランディング → アイテム選択 → 日本語で要望入力 → 要件確認 → 型紙生成 → 寸法微調整 → PDF ダウンロード（ここまでログインなしでも 1 回体験可）

### デザイン要件

- 革・ハンドメイド・温もりを感じさせるトーン（アースカラー、セリフ系見出し、革のテクスチャを差し色に）
- 初心者でも迷わないアイコン主体のナビゲーション
- 重要な操作（生成・ダウンロード）は常に画面内に固定で見えること

### レスポンシブ

- スマホ（375px〜）: チャットとプレビューはタブ切り替え
- タブレット（768px〜）: 2 ペイン構成、サイドバーは折りたたみ
- デスクトップ（1024px〜）: 3 領域（チャット / プレビュー / 寸法調整）並列表示

## 制約事項

### スコープ外（MVP では実装しない）

- **対応アイテムの拡張**: ベルト、ブックカバー、名刺入れ、トートバッグ、バッグ類は対象外（ロードマップ 2027 Q1 以降）
- **コミュニティ・型紙マーケット**: 型紙の公開・販売・共有機能は v2.0
- **3D プレビュー**: 完成形の 3D レンダリングは対象外
- **金具データベース連携**: ファスナー・ホック・カシメの型番別寸法データ連携は対象外
- **多言語対応**: 英語 UI は対象外（日本語のみ）
- **モバイルアプリ**: ネイティブアプリは対象外（PWA 対応のみ検討可）
- **法人向け機能**: チームアカウント、複数ユーザー管理、SSO、API 連携は Pro / Enterprise プラン向けで MVP では未実装
- **手書きスケッチ入力**: 画像 / 手書きからの型紙生成は対象外（チャットテキストのみ）
- **革素材の販売連携**: 必要な革を購入できる EC 連携は対象外

### 既知の制限

- AI 生成型紙はプロ職人の手作業設計を完全に代替するものではなく、初心者〜中級者の補助ツールとして提供する
- 極端な寸法（例: 名刺サイズ未満、A3 を超える大型）は生成精度が保証されない
- 革種・厚み・伸縮特性のすべてを完全に考慮することは困難であり、ユーザー側での試作・調整を前提とする
- 印刷時の精度はプリンタの設定（実寸スケール、フチなし設定）に依存する

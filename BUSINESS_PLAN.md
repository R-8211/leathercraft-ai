> 作成日: 2026-05-07

---

## エグゼクティブサマリー

**LeatherCraft AI** は、AIを活用した革小物の型紙自動生成Webアプリケーションである。ユーザーが日本語で作りたいアイテムを説明するだけで、カスタム寸法のSVG型紙を即座に生成・ダウンロードできる。革職人・DIY愛好者・ハンドメイド販売者をターゲットに、従来は専門知識が必要だった型紙設計を民主化する。

## 1. 市場調査

### 1.1 国内ハンドメイド市場規模

| 指標 | 数値 | 備考 |
|------|------|------|
| 国内ハンドメイド市場規模（2024年） | 約2,300億円 | 矢野経済研究所推計 |
| 年間成長率（CAGR） | 8〜12% | コロナ禍以降の巣ごもり需要継続 |
| レザークラフト人口（推計） | 約120万人 | 日本国内の愛好者・趣味層 |
| ハンドメイドECの流通総額（Creema+minne） | 約750億円 | 2024年実績 |
| そのうち革製品カテゴリ比率 | 約15〜18% | 財布・バッグ・キーケース等 |

### 1.2 競合分析

#### 直接競合（型紙生成・販売サービス）

| サービス名 | 特徴 | 弱点 |
|-----------|------|------|
| Creema 型紙販売 | 人力作成の型紙を個人販売 | カスタマイズ不可、購入コスト発生 |
| minne パターン | 同上 | 同上 |
| Adobe Illustrator | 汎用ベクターツール | 専門知識が必要、高額（月3,280円〜） |
| CorelDRAW / Affinity | 同上 | 同上 |
| 海外：Sewing Pattern Generator | 衣類特化、英語のみ | 革小物未対応 |

#### 間接競合（AI関連クラフトツール）

| サービス名 | 特徴 | 弱点 |
|-----------|------|------|
| Canva AI | デザイン生成 | 型紙・実寸出力機能なし |
| ChatGPT + Code | テキスト指示でコード生成 | UI不在、技術知識必須 |
| 手芸ナビ | レシピ提供 | 静的コンテンツ、AI非活用 |

### 1.3 ターゲットユーザーセグメント

#### セグメント A：入門〜中級レザークラフター（主力）
- **人口**: 推計60〜80万人
- **特徴**: 趣味として楽しんでいるが、型紙設計に苦手意識
- **ニーズ**: 自分の好みに合わせたサイズの型紙がすぐほしい
- **支払い意欲**: 月額500〜1,500円程度

#### セグメント B：ハンドメイド販売者（高単価）
- **人口**: 推計10〜15万人（Creema/minne出品者）
- **特徴**: 受注制作でカスタムサイズ対応が必要
- **ニーズ**: 受注ごとに最適な型紙を素早く生成したい
- **支払い意欲**: 月額2,000〜5,000円程度

#### セグメント C：レザークラフト教室・スクール（法人）
- **人口**: 全国推計2,000〜3,000教室
- **特徴**: 生徒向けの型紙準備が定期的に必要
- **ニーズ**: オリジナル教材の型紙を効率的に作成
- **支払い意欲**: 月額5,000〜20,000円程度（法人契約）

### 1.4 市場トレンド

1. **AIツール普及加速**: ChatGPT登場以降、非エンジニア層のAI活用が急増。自然言語でのツール操作への受容度が高まっている。
2. **ハンドメイドECの成長**: minne・Creemaの登録者数増加が続き、副業・小遣い稼ぎ目的の制作者が増加。
3. **SVG・PDF型紙の需要増**: A4サイズでの家庭用プリンター出力ニーズが高まっており、デジタル型紙市場が拡大中。
4. **サステナビリティ意識**: 長く使える本革製品への関心増加。革製品DIYへの関心が連動して上昇。
5. **動画コンテンツとの連動**: YouTubeでのレザークラフト動画が人気を博しており、視聴者が実際に制作するフローに乗れるツールの需要がある。

## 2. プロダクト概要

### 2.1 コアバリュープロポジション

> 「作りたいものを日本語で伝えるだけで、プロ品質のレザー型紙が30秒で手に入る」

### 2.2 主要機能

| 機能 | 説明 | 優先度 |
|------|------|--------|
| AIチャット型紙生成 | 自然言語入力 → Claude API → SVG型紙出力 | ★★★ MVP |
| リアルタイムSVGプレビュー | 寸法調整時に即時反映 | ★★★ MVP |
| PDFダウンロード | A4実寸印刷対応、縫い代・切り込み線付き | ★★★ MVP |
| 寸法パラメータ調整 | スライダーでサイズを微調整 | ★★☆ v1.1 |
| 型紙ライブラリ保存 | 過去の型紙を保存・再利用 | ★★☆ v1.1 |
| 材料計算機能 | 必要な革の面積・コストを算出 | ★☆☆ v1.2 |
| コミュニティ共有 | 型紙を公開・販売できるマーケット | ★☆☆ v2.0 |

### 2.3 対応アイテム（MVP）

- **財布系**: 二つ折り財布、三つ折り財布、長財布、カードケース
- **小物系**: 小銭入れ、キーケース、コインケース
- **今後拡張**: ベルト、ブックカバー、名刺入れ、トートバッグ

### 2.4 技術スタック

```
Frontend:  React 18 + Vite + TypeScript + Tailwind CSS
Backend:   Node.js + Express + Claude API (claude-sonnet-4-6)
SVG生成:   カスタムアルゴリズム + SVG.js
PDF出力:   jsPDF / Puppeteer
DB:        PostgreSQL（将来）/ ファイルベース（MVP）
```

## 6. リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| Claude API コスト増大 | 高 | キャッシュ活用・プロンプト最適化・ユーザー回数制限 |
| 型紙精度への不満 | 高 | ベータ期間中に職人フィードバックで調整 |
| 大手参入（Adobe AI等） | 中 | 日本語・革特化・コミュニティで差別化 |
| 著作権・知財問題 | 中 | 生成型紙の利用規約を明確化 |
| ユーザー離脱率 | 中 | 保存・共有機能でエンゲージメント向上 |

## 9. まとめ

LeatherCraft AI は、日本国内120万人のレザークラフト愛好者が抱える「型紙作成の専門知識・時間的障壁」を、AIの力で一挙に解決するプロダクトである。MVPを迅速に構築し、ユーザーフィードバックをもとに反復改善することで、2027年末までにARR 1億円を射程圏に入れることを目標とする。
